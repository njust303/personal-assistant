## 一、设计概述

### 1.1 功能定位

市场看板是个人理财助手的**核心数据展示模块**，提供：
- 实时股票行情（A 股/港股/美股）
- 每日基金净值
- 黄金/原油/汇率等其他数据

### 1.2 技术选型

| 组件 | 技术方案 | 说明 |
|------|---------|------|
| 数据存储 | 飞书多维表格 | 结构化数据，支持筛选排序 |
| 数据展示 | 飞书表格视图 | 原生界面，零开发成本 |
| 数据更新 | OpenClaw Cron + API | 定时抓取，自动更新 |
| 推送通知 | 飞书群消息 | Markdown 格式，简洁清晰 |

## 四、数据更新策略

### 4.1 更新频率

| 数据类型 | 更新频率 | 更新时间 | 数据源 |
|---------|---------|---------|--------|
| A 股行情 | 5 分钟 | 交易时段 9:30-15:00 | 新浪财经 API |
| 港股行情 | 5 分钟 | 交易时段 9:30-16:00 | 新浪财经 API |
| 美股行情 | 5 分钟 | 交易时段 21:30-4:00 | Yahoo Finance API |
| 基金净值 | 每日 | 交易日 20:00 | 天天基金 API |
| 黄金/原油 | 每小时 | 整点 | 新浪财经 API |
| 汇率 | 每日 | 09:00 | 中国银行 API |

### 4.2 更新流程

```python
# 伪代码示例
def update_market_data():
    # 1. 获取当前时间
    current_time = get_current_time()
    
    # 2. 判断市场状态
    if is_trading_time('A 股'):
        # 3. 调用 API 获取数据
        stock_data = fetch_from_sina_api()
        
        # 4. 数据清洗
        cleaned_data = clean_data(stock_data)
        
        # 5. 更新飞书表格
        for record in cleaned_data:
            feishu_bitable_update_record(
                app_token='EIS3beotkaQ8BcsqafNcRzQbnSg',
                table_id='tblgZZHOZ6HkDa9B',
                record_id=record['id'],
                fields={
                    'current_price': record['price'],
                    'change_percent': record['change'],
                    'update_time': current_time
                }
            )
        
        # 6. 记录日志
        log_update_result('股票行情', len(cleaned_data))
```

### 4.3 异常处理

| 异常类型 | 处理策略 |
|---------|---------|
| API 超时 | 重试 3 次，间隔 5 秒 |
| 数据异常 | 跳过该条记录，记录告警 |
| 飞书 API 失败 | 重试 3 次，记录日志 |
| 市场休市 | 不更新，保持上次数据 |
