## 一、系统概述

### 1.1 项目背景

个人理财助手是一个轻量级理财信息平台，帮助用户：
- 学习理财知识
- 了解市场动态
- 获取平台推荐
- 接收理财资讯

### 1.2 核心目标

| 目标 | 指标 | 优先级 |
|------|------|--------|
| 知识普及 | 提供 100+ 篇理财基础知识 | 🔴 P0 |
| 信息聚合 | 整合 5+ 个数据源 | 🔴 P0 |
| 平台推荐 | 评分 10+ 个理财平台 | 🔴 P0 |
| 定时推送 | 每日准时推送资讯 | 🔴 P0 |

## 二、系统架构

### 2.1 整体架构

```
┌─────────────────────────────────────────┐
│           用户层                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │ 飞书群  │  │ 飞书文档│  │  邮箱   │ │
│  └─────────┘  └─────────┘  └─────────┘ │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           应用层                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │知识库   │  │市场看板 │  │平台推荐 │ │
│  │模块     │  │模块     │  │模块     │ │
│  └─────────┘  └─────────┘  └─────────┘ │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           数据层                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │飞书多维 │  │本地文件 │  │外部 API │ │
│  │表格     │  │存储     │  │数据源   │ │
│  └─────────┘  └─────────┘  └─────────┘ │
└─────────────────────────────────────────┘
```

### 2.2 技术选型

| 模块 | 技术 | 说明 |
|------|------|------|
| **前端展示** | 飞书文档/飞书群 | 用户交互界面 |
| **数据存储** | 飞书多维表格 | 结构化数据存储 |
| **内容生成** | OpenClaw + AI | 内容创作和整理 |
| **定时任务** | OpenClaw Cron | 定时推送 |
| **邮件发送** | QQ 邮箱 SMTP | 邮件推送 |

## 三、模块设计

### 3.1 知识库模块

**功能：**
- 理财基础知识（100+ 篇）
- 政策解读（实时更新）
- 投资技巧（持续更新）

**数据结构：**
```markdown
知识库文章
├── 标题
├── 分类（入门/进阶/高级）
├── 标签（基金/股票/保险等）
├── 内容
├── 来源链接
├── 创建时间
└── 更新时间
```

**存储方式：** 飞书文档

---

### 3.2 市场看板模块

**功能：**
- A 股/港股/美股行情
- 基金净值
- 黄金/原油价格
- 汇率数据

**数据源：**
| 数据类型 | 数据源 | 更新频率 |
|---------|--------|---------|
| A 股行情 | 新浪财经 API | 实时 |
| 港股行情 | 新浪财经 API | 实时 |
| 美股行情 | Yahoo Finance | 每日 |
| 基金净值 | 天天基金 API | 每日 |
| 黄金/原油 | 新浪财经 | 每小时 |
| 汇率 | 中国银行 API | 每日 |

**存储方式：** 飞书多维表格

---

### 3.3 平台推荐模块

**功能：**
- 理财平台评分
- 产品推荐
- 风险提示

**评分维度：**
| 维度 | 权重 | 说明 |
|------|------|------|
| 安全性 | 30% | 平台资质、资金托管 |
| 收益率 | 25% | 平均收益水平 |
| 流动性 | 20% | 赎回便捷性 |
| 用户体验 | 15% | APP 体验、客服 |
| 费用成本 | 10% | 手续费、管理费 |

**推荐平台：**
1. 支付宝（蚂蚁财富）
2. 微信（理财通）
3. 天天基金
4. 蛋卷基金
5. 招商银行 APP
6. 华泰证券 APP

**存储方式：** 飞书多维表格

## 四、数据流设计

### 4.1 知识库数据流

```
内容收集 → 内容整理 → 飞书文档 → 用户阅读
    ↓          ↓          ↓          ↓
 搜索      AI 整理    存储      飞书群链接
```

### 4.2 市场数据流

```
外部 API → 数据清洗 → 飞书多维表格 → 用户查看
   ↓          ↓            ↓            ↓
 定时抓取   格式转换    存储       飞书群链接
```

### 4.3 推送数据流

```
定时任务 → 内容生成 → 飞书群/邮箱 → 用户接收
   ↓           ↓           ↓            ↓
  Cron     AI 整理    多渠道       定时接收
```

## 六、安全设计

### 6.1 数据安全

- 不存储用户敏感信息（银行卡、密码等）
- 所有数据公开可查
- 免责声明明确

### 6.2 合规设计

- 不提供具体投资建议
- 所有数据注明来源
- 风险提示醒目展示
