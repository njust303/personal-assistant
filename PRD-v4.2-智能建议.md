## 三、智能建议功能详细说明

### 3.1 功能描述
基于真实市场数据，每日自动生成投资建议，包括：
- 📊 市场观点
- ⚖️ 资产配置
- 🎯 操作策略

### 3.2 数据来源
| 数据项 | 来源 | 说明 |
|--------|------|------|
| 上证指数 | 腾讯财经 API | 实时行情 |
| 深证成指 | 腾讯财经 API | 实时行情 |
| 创业板指 | 腾讯财经 API | 实时行情 |
| PE 估值 | 历史数据统计 | 估值分位 |
| PB 估值 | 历史数据统计 | 估值分位 |

### 3.3 生成逻辑

#### 3.3.1 市场情绪判断
```python
平均涨跌幅 = (上证涨跌幅 + 深证涨跌幅 + 创业板涨跌幅) / 3

if 平均涨跌幅 > 2%:
    市场状态 = '火热'
elif 平均涨跌幅 > 0.5%:
    市场状态 = '温和上涨'
elif 平均涨跌幅 > -0.5%:
    市场状态 = '震荡整理'
elif 平均涨跌幅 > -2%:
    市场状态 = '调整'
else:
    市场状态 = '低迷'
```

#### 3.3.2 估值水平判断
```python
平均 PE 分位 = (上证 PE 分位 + 深证 PE 分位 + 创业板 PE 分位) / 3

if 平均 PE 分位 < 20%:
    估值水平 = '极低'
elif 平均 PE 分位 < 40%:
    估值水平 = '较低'
elif 平均 PE 分位 < 60%:
    估值水平 = '合理'
elif 平均 PE 分位 < 80%:
    估值水平 = '较高'
else:
    估值水平 = '极高'
```

#### 3.3.3 市场观点生成
根据市场状态和估值水平，从模板库中选择合适的观点：

| 市场状态 | 估值水平 | 观点示例 |
|---------|---------|---------|
| 火热 | 极高 | 市场情绪高涨，注意风险控制 |
| 火热 | 合理 | 赚钱效应显著，保持理性 |
| 震荡整理 | 较低 | 板块轮动加快，逢低布局 |
| 调整 | 较低 | 调整不改长期趋势，估值吸引力提升 |
| 低迷 | 极低 | 底部区域，长期投资者可积极 |

#### 3.3.4 资产配置建议
根据估值水平调整股票、债券、黄金的比例：

| 估值水平 | 股票比例 | 债券比例 | 黄金比例 |
|---------|---------|---------|---------|
| 极低 | 70-75% | 15-25% | 5-10% |
| 较低 | 65-70% | 20-25% | 10% |
| 合理 | 55-60% | 30-35% | 10% |
| 较高 | 45-50% | 40-45% | 10% |
| 极高 | 30-40% | 50-60% | 10% |

#### 3.3.5 操作策略生成
根据市场状态和估值水平生成操作建议：

| 市场状态 | 估值水平 | 策略示例 |
|---------|---------|---------|
| 火热 | 极高 | 适度止盈，控制仓位 |
| 温和上涨 | 合理 | 逢低布局，避免追高 |
| 震荡整理 | 合理 | 观望为主，等待方向 |
| 调整 | 较低 | 逢低布局，积累筹码 |
| 低迷 | 极低 | 积极定投，积累便宜筹码 |

### 3.4 更新机制
- **自动更新**: 每日凌晨 6 点 Cron 自动执行
- **手动刷新**: 页面点击"🔄 刷新"按钮
- **历史归档**: 保存每日建议，保留 30 天

### 3.5 数据格式
```json
{
  "date": "2026-03-15",
  "update_time": "2026-03-15 06:00:00",
  "market_data": {
    "sh000001": {"name": "上证指数", "price": 4095.45, "change": -0.54},
    "sz399001": {"name": "深证成指", "price": 14280.78, "change": -0.14},
    "sz399006": {"name": "创业板指", "price": 3310.28, "change": 0.42}
  },
  "sentiment": {
    "state": "neutral",
    "state_name": "震荡整理"
  },
  "valuation": "合理",
  "advice": {
    "market_view": "板块轮动加快，注意节奏",
    "asset_allocation": "股票 60% + 债券 30% + 黄金 10%",
    "operation_strategy": "观望为主，等待方向"
  }
}
```

## 五、技术实现

### 5.1 技术栈
| 模块 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML5 + CSS3 + JavaScript | 响应式设计 |
| 后端 | Python 3.6+ | 脚本处理 |
| 数据处理 | Pandas (可选) | 数据分析 |
| 定时任务 | Cron | 每日自动执行 |
| 数据存储 | JSON 文件 | 本地存储 |

### 5.2 文件结构
```
h5-app/
├── index.html                    # 主页面
├── fetch_news.py                 # 新闻抓取脚本
├── generate_advice.py            # 建议生成脚本
├── refresh_news.sh               # 手动刷新脚本
├── news_data/                    # 新闻数据目录
│   ├── news_today.json          # 当日新闻
│   ├── news_YYYY-MM-DD.json     # 历史归档
│   └── update.log               # 更新日志
└── advice_data/                  # 建议数据目录
    ├── advice_today.json        # 当日建议
    ├── advice_YYYY-MM-DD.json   # 历史归档
    └── update.log               # 更新日志
```

### 5.3 Cron 配置
```bash
# 每日凌晨 6 点更新新闻
0 6 * * * cd /home/admin/.openclaw/workspace/projects/个人理财助手/h5-app && python3 fetch_news.py >> news_data/update.log 2>&1

# 每日凌晨 6 点更新建议
0 6 * * * cd /home/admin/.openclaw/workspace/projects/个人理财助手/h5-app && python3 generate_advice.py >> advice_data/update.log 2>&1
```
