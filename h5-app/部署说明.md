## ⚠️ 重要提示

### 数据更新方式

由于 GitHub Pages 是静态托管，**数据不会自动更新**。需要：

**方式 1: 手动刷新**
- 在页面点击 "🔄 刷新" 按钮
- 或刷新浏览器页面

**方式 2: 本地运行脚本（推荐）**
```bash
cd /home/admin/.openclaw/workspace/projects/个人理财助手/h5-app
python3 fetch_stock_data.py
git add stock_data.json
git commit -m "data: 更新股票数据"
git push github main
git push gitee main
```

**方式 3: 配置 GitHub Actions（自动化）**
创建 `.github/workflows/update-data.yml` 定时更新数据
