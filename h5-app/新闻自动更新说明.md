## ⏰ 自动更新配置

### Cron 任务
```bash
# 每天凌晨 6 点自动更新
0 6 * * * cd /home/admin/.openclaw/workspace/projects/个人理财助手/h5-app && python3 fetch_news.py >> news_data/update.log 2>&1
```

### 查看 Cron 任务
```bash
crontab -l | grep fetch_news
```

### 修改更新时间
```bash
# 编辑 Cron
crontab -e

# 修改时间（例如改为早上 8 点）
0 8 * * * cd /home/admin/.openclaw/workspace/projects/个人理财助手/h5-app && python3 fetch_news.py >> news_data/update.log 2>&1
```

## 📊 数据格式

### news_today.json
```json
{
  "date": "2026-03-15",
  "update_time": "2026-03-15 13:15:07",
  "news": {
    "finance": [
      {
        "t": "新闻标题",
        "s": "来源",
        "time": "10 分钟前",
        "u": "链接",
        "hot": true
      }
    ],
    "ai": [...],
    "tea": [...],
    "global": [...]
  }
}
```

### 字段说明
- `t`: 新闻标题 (title)
- `s`: 来源 (source)
- `time`: 发布时间
- `u`: 链接 (URL)
- `hot`: 是否热门 (true/false)

## 🗂️ 历史归档

### 归档规则
- **文件名**: `news_YYYY-MM-DD.json`
- **保留时间**: 30 天
- **自动清理**: 超过 30 天的归档自动删除

### 查看历史
```bash
# 查看所有归档
ls -lh news_data/news_*.json

# 查看特定日期
cat news_data/news_2026-03-10.json
```

## 🔍 常见问题

### Q1: 新闻不更新怎么办？

**A**: 检查 Cron 任务和日志
```bash
# 检查 Cron
crontab -l | grep fetch_news

# 检查日志
cat news_data/update.log
```

### Q2: 如何手动更新？

**A**: 运行脚本
```bash
python3 fetch_news.py
```

### Q3: 如何修改新闻来源？

**A**: 编辑 `fetch_news.py` 中的 `CATEGORIES` 字典

### Q4: 如何增加新闻数量？

**A**: 编辑 `fetch_news.py` 中的 `templates` 字典，添加更多模板
